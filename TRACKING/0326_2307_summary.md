# 开发总结 2026-03-26 23:07

## 变更清单

### 新增文件

- **`test.py`** — 串口协议自动化测试脚本
  - 依赖：`pyserial`
  - 用法：`python test.py [COM口]`，默认 COM3
  - 包含 4 个测试用例：
    1. 心跳帧：发送 `CMD=0x00`，验证原样回传、SEQ 一致
    2. CRC 错误帧：故意填错 CRC，验证响应 `CMD=0x01`、`SEQ=0xFF`、错误码 `0x01`
    3. 未知命令码 `0x10`：验证响应 `CMD=0x01`、错误码 `0x02`
    4. 连续两帧：验证状态机响应后正确复位
  - 全部通过时输出 `4 通过 / 0 失败`，退出码 0
