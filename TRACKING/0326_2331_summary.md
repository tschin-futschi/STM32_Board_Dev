## 变更清单

### 修改文件

- **`App/Src/app_protocol.c`** — 重构命令分发逻辑
  - 新增 `DispatchFrame(const Proto_Frame_t *pFrame)` 函数，将命令码分发的 `switch-case` 从 `ProcessByte()` 中提取出来
  - `STATE_WAIT_CRC_L` 从 20 行缩减至 4 行有效逻辑，只负责 CRC 验证和调用 `DispatchFrame()`
  - 后续新增命令只需在 `DispatchFrame()` 里加 `case`，`ProcessByte()` 不需要改动

### 重构前后对比

**重构前（STATE_WAIT_CRC_L）：**
```c
if (rxCrc != s_rxCrcAccum)
    SendErrorResp(PROTO_CRC_ERR_SEQ, PROTO_ERR_CRC_FAIL);
else {
    switch (s_rxFrame.cmd) {
        case PROTO_CMD_HEARTBEAT: HandleHeartbeat(&s_rxFrame); break;
        default: SendErrorResp(s_rxFrame.seq, PROTO_ERR_UNKNOWN_CMD); break;
    }
}
```

**重构后（STATE_WAIT_CRC_L）：**
```c
if (rxCrc != s_rxCrcAccum)
    SendErrorResp(PROTO_CRC_ERR_SEQ, PROTO_ERR_CRC_FAIL);
else
    DispatchFrame(&s_rxFrame);
```
