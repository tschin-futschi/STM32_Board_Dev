## 阶段 1 代码结构与功能详解

### 系统上电启动流程

```
上电/复位
  → startup_stm32f429_439xx.s (Reset_Handler)
    → 从 Flash 复制 .data 到 SRAM
    → 清零 .bss
    → 调用 SystemInit()          ← system_stm32f4xx.c（配置时钟树 HSE 8MHz → 180MHz）
    → 调用 main()                ← main.c
      → NVIC 分组(Group_4)
      → SysTick 1ms 初始化
      → LED GPIO 初始化
      → while(1) 500ms 心跳循环
```

### 关键设计点

1. **非阻塞心跳**：`now - prevTick >= 500` 时间差判断，uint32_t 减法天然处理溢出
2. **ISR 最小化**：SysTick_Handler 仅递增 `s_tickMs`，不做任何阻塞操作
3. **volatile 保护**：`s_tickMs` 在 ISR 和主循环间共享，volatile 防止编译器优化
4. **RCC 先行**：BSP_LED_Init() 第一步 `RCC_AHB1PeriphClockCmd`，符合 SPL 规范
5. **硬件参数隔离**：端口/引脚/时钟全在 bsp_led.h 宏中，.c 文件无裸写数字
6. **-pipe 标志**：解决 Windows 环境下 GCC 临时文件 `C:\WINDOWS\` 权限拒绝问题
