## 协议 v1.4 完整命令码汇总

### 系统控制组（0x00~0x1F）

| 命令码 | 名称 | 方向 |
|--------|------|------|
| `0x00` | 心跳 | 双向 |
| `0x01` | 错误响应 | STM32→PC |
| `0x02` | 设置电机 IC 地址 | PC→STM32 |
| `0x03` | 设置波特率 | PC→STM32 |
| `0x04` | 系统复位 | PC→STM32 |
| `0x05` | 电机应答测试 | PC→STM32 |
| `0x06` | 调试信息 | STM32→PC |

### 寄存器读写组（0x20~0x4F）

| 命令码 | 名称 | 方向 |
|--------|------|------|
| `0x20` | 读单个寄存器 | PC→STM32 |
| `0x21` | 写单个寄存器 | PC→STM32 |
| `0x22` | 批量读寄存器（分包返回） | PC→STM32 |

### 示波器控制组（0x50~0x7F）

| 命令码 | 名称 | 方向 |
|--------|------|------|
| `0x50` | 启动采样 | PC→STM32 |
| `0x51` | 停止采样 | PC→STM32 |
| `0x52` | 设置采样间隔 | PC→STM32 |
| `0x53` | 设置采样通道 | PC→STM32 |
| `0x54` | 设置通道寄存器映射 | PC→STM32 |

### 关键设计决策

- 数据流帧新增 LEN 字段：`[0xBB][掩码][LEN][数据][XOR]`
- `0x02` 失败时 `g_motorIcAddr` 保持原值不变
- `0x03` 用旧波特率 echo，再切换
- `0x04` 复位前上位机必须先切回 115200
- `0x22` 分包期间忽略所有控制帧；采样中收到返回错误
- `0x51` 立即停止并清空缓存
- 启动采样失败时发 `0x06` 说明原因
- 默认采样间隔 1000us，通道掩码 0x01，映射全 0xFFFF
